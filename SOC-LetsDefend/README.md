# SOC-LetsDefend

A personal SOC analyst documentation portfolio built from hands-on practice 
on the LetsDefend blue team training platform.

## What This Is
Structured writeups for every alert I investigate, following real SOC 
documentation standards including triage, IOC analysis, MITRE ATT&CK mapping, 
and remediation recommendations.

## Structure
- **Alerts/** — individual ticket writeups organized by category
  - Phishing
  - Malware
  - Network
  - Endpoint
- **Playbooks/** — reusable SOPs for each alert type

## Platform
[LetsDefend](https://letsdefend.io) — Blue Team Training

## Skills Demonstrated
- Alert triage and investigation
- IOC identification and analysis
- MITRE ATT&CK framework mapping
- SOC documentation and reporting
- Threat intelligence gathering
